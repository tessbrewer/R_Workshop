# Basic data manipulation in R
Here we'll go over the basics of the basics in R. This tutorial focuses mainly on functions and terms we'll use later on. **You should not need any experience in R to follow along.**

Topics covered:  
 * Setting working directories and relative vs absolute paths
 * Loading and installing packages
 * Basic syntax and operators
 * Loading data and accessing elements of a dataframe
 * Subsetting your dataframe
 * Modifying dataframes
 * Saving your files
